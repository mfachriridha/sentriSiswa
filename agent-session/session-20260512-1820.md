## Architectural Decisions
- **Brand color:** `#1c6880` (school official blue), hover `#155066`, light `#d4e8ee`
- **No dark mode** — explicitly disabled
- **Font:** Inter (300-900 weights)
- **Icons:** Bootstrap Icons v1.11.3 (CDN)
- **Tailwind:** v4 via CDN fallback (`@tailwindcss/browser@4`) when Vite manifest missing
- **CSS variables:** Defined in both `app.css` `@theme` and inline `<style>` fallbacks in layouts
- **Component pattern:** `@extends('layouts.app')` for authenticated, `@extends('layouts.guest')` for public
- **Toast system:** Centralized via `components/alert-toast.blade.php` — auto-displays Laravel session flashes
- **Loading overlay:** Centralized via `components/loading-overlay.blade.php` — auto-triggers on form submit
- **Input base font:** `.875rem` (14px), minimum `text-sm` for labels/body content
- **`untuk-refrensi/` folder:** NEVER delete — contains reference images and CSV samples

## Critical Snippets

### Toast API (global, included via layout)
```js
showToast(message, type = 'info', duration = 4000)
// types: 'success' | 'error' | 'warning' | 'info'
```

### Loading API (global, included via layout)
```js
showLoading('Custom message...')  // Shows overlay, disables submit buttons
hideLoading()                       // Hides overlay, re-enables buttons
// Auto-triggers on <form> submit unless form has data-noLoading attribute
```
